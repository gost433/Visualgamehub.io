Guia de Contribuição | Vanguards Visual Studio
Primeiramente, obrigado por se interessar em contribuir com os projetos da VisualStudioAndSecurity. Como uma iniciativa focada em Software de Alta Performance, Design Estratégico e Cybersecurity, mantemos padrões rígidos para garantir a integridade e a qualidade de cada linha de código.

Ao contribuir, você concorda em seguir este Código de Conduta e as diretrizes técnicas abaixo.

0x01. Padrões de Código (Code Standards)
Para manter a consistência entre nossos projetos (seja em sistemas de trading, automação ou ferramentas visuais), exigimos:

Clean Code: Variáveis e funções devem ter nomes semânticos e claros.

Documentação Inline: Comente blocos complexos, especialmente em algoritmos de trading ou lógica de criptografia.

Commits Semânticos: Utilize o padrão Conventional Commits (ex: feat:, fix:, docs:, refactor:).

0x02. Protocolo de Segurança (Cybersecurity First)
Como especialistas em segurança, não aceitamos contribuições que exponham vulnerabilidades.

Segredos e Chaves: Nunca envie chaves de API, segredos de ambiente (.env) ou credenciais no código.

Dependências: Antes de adicionar uma nova biblioteca, verifique se ela é estável e não possui vulnerabilidades conhecidas (CVEs).

Sanitização: Todo input de usuário deve ser tratado e sanitizado para evitar ataques de injeção.

0x03. Processo de Pull Request (PR)
Fork o Repositório: Crie uma cópia do projeto em sua conta.

Crie uma Branch: Use nomes descritivos (ex: feature/nova-funcionalidade ou patch/correcao-bug).

Testes: Certifique-se de que sua alteração não quebra funcionalidades existentes (especialmente o sistema de UI/UX e proteções de código).

Submeta o PR: Descreva detalhadamente o que foi alterado e por que essa alteração é necessária.

0x04. Identidade Visual e UI
Se sua contribuição envolver a interface (Frontend/Design), siga a estética da Vanguards Visual:

Paleta de cores: Dark Mode (High Contrast).

Tipografia: Inter ou JetBrains Mono para elementos técnicos.

Foco em performance e carregamento instantâneo.
