Ameaça Crítica: FAUX#ELEVATE

Relatório de Inteligência: Mineração e Exfiltração de Dados

Hackers estão utilizando uma nova abordagem para roubar credenciais empresariais e implementando o Crypto miner (Mecanismo de Mineração de Crypto ativos).

Há uma campanha rodando com o foco em ambientes corporativos na qual currículos falsos implantados com um mecanismo de mineradores e roubo de informações utilizando VBScript com ofuscação disfarçados de Doc de currículos/Cv, os cibercriminosos utilizam E-mails de Phishing, quando ativo, o vírus/malware aplica uma execução de ferramentas multiuso configurada para extrair dados e minerar crypto ativos da rede Monero.

O mecanismo se tornou popularmente chamado de Faux#Elevate pelas instituições de defesa cibernéticas, a campanha se destaca por seus abusos de serviços e infraestruturas de legítimos, utilizando DropBox para o armazenamento de payloads, sites como Wordpress marrocos para hospedarem configurações de controle (C2) E E-MAIL. Também utilizam ru SMTP para exfiltração das credenciais dos navegadores alvos e arquivos de Desktop, exemplo esse que nos leva a enxergar como os ataques podem enganar os mecanismos de segurança sem chamarem muito atenção.

Os invasores utilizam um sistema de Visual Basic Script que consiste em enviar e exibir uma mensagem de erro, na qual os destinatários acham que o arquivo corrompeu, porém, não imaginam que a mensagem é só um "Bait", uma forma de "isca" para o script com 224 mil linhas, na qual somente 226 linhas contêm o código real, executando várias verificações, evitando sandboxes e entrando em looping de controle persistente da conta do destinatário/Usuário. O malware também tem um mecanismo de porta de união de domínio, utilizando Windows Management Instrumentation, visando apenas os sistemas empresariais. Assim que o atacante entra na máquina da empresa, desativa os controles de segurança para camuflar seus rastros, configurando e excluindo o Microsoft Defender, desativando o UAC por meio de alterações no registros do Windows e se auto-excluindo.

Entre as ferramentas usadas para facilitar o roubo de credenciais está um componente que utiliza o projeto ChromElevator para extrair dados sensíveis de navegadores baseados em Chromium, contornando as proteções contra criptografia vinculada a aplicativos (ABE). Algumas das outras ferramentas incluem:

mozilla.vbs: um malware VBScript para roubo do perfil e credenciais do Mozilla Firefox
walls.vbs: um payload VBScript para exfiltração de arquivos em desktop
mservice.exe: um minerador de criptomoedas XMRig lançado após recuperar a configuração de mineração de um site marroquino do WordPress comprometido
WinRing0x64.sys: um driver legítimo do kernel do Windows usado para desbloquear todo o potencial de mineração da CPU
RuntimeHost.exe: um componente de Trojan persistente que modifica as regras do Firewall do Windows e se comunica periodicamente com um servidor C2

Os dados exclusivos do navegador são exfiltrados usando dois e-mails separados de contas remetentes ru ("olga.aitsaid@mail.ru" e "3pw5nd9neeyn@mail.ru") que compartilham a mesma senha via SMTP para outro endereço de e-mail operado pelo ator ameaçador ("vladimirprolitovitch@duck.com").

Uma vez concluídas as atividades de roubo e exfiltração de credenciais, a cadeia de ataque inicia uma limpeza agressiva de todas as ferramentas caídas na tentativa de minimizar a pegada forense, deixando apenas o minerador e o troiano.

"A campanha FAUX#ELEVATE demonstra uma operação de ataque bem organizada e em múltiplas etapas que combina várias técnicas notáveis em uma única cadeia de infecção", disse Securonix. "O que torna esta campanha particularmente perigosa para as equipes de segurança corporativas é a velocidade de execução, a cadeia completa de infecção completa em aproximadamente 25 segundos desde a execução inicial do VBS até a exfiltração de credenciais, e o direcionamento seletivo de máquinas conectadas ao domínio, que garante que cada host comprometido ofereça o máximo valor por meio do roubo de credenciais corporativas e sequestro persistente de recursos."

_terminologia_técnica

WMI (Windows Management Instrumentation)

Infraestrutura da Microsoft para gerenciamento de dados e operações em sistemas operacionais Windows. No caso do FAUX#ELEVATE, é explorada para persistência silenciosa e coleta de dados sem arquivos físicos.

ABE (App-Bound Encryption)

Camada de proteção que vincula segredos (como senhas salvas) à identidade de um executável específico no Chrome. O malware utiliza o ChromElevator para tentar contornar essa proteção.
